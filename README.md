# mini-redis

in-memory key-value хранилище со своим сетевым протоколом

Что реализовано:
- сетевое программирование на уровне TCP
- паттерн горутина на соединение
- конкурентный доступ к общим данным: sync.RWMutex
- фоновые задачи: горутина-очиститель с time.Ticker
- разбиение на пакеты с единственной ответственностью
- тесты, Docker

В одном терминале:
- go run . -port 6379 / docker run -p 6379:6379 mini-redis

В другом терминале:
- nc localhost 6379

Пример использования:
SET name Anna
OK

GET name
Anna

SET token abc123 EX 5
OK
